This page contains analyses of the mobility data released by Facebook during the Covid-19 crisis.
This analysis uses data provided by Facebook, under the Facebook Data For Good initiative. This document contains all the code necessary to repeat this analysis. However, you will need to have access to the Facebook Data For Good. More information is available here.


Please click on a country to see the analysis.
